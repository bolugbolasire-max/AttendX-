🔒 Force Logout a User

Marks the account as disabled, which blocks their next login attempt. This does not immediately kill an already-open browser session on their device — Firebase Auth has no built-in "kill session now" for client SDKs — but it will stop them from logging back in, and your app's own auth guard checks account status on every page load.
